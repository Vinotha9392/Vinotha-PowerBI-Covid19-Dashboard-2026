# COVID-19 Dashboard using Power BI

## Project Overview
Developed an interactive COVID-19 Dashboard in Power BI to analyze and visualize global pandemic trends.
The dashboard provides meaningful insights into worldwide COVID-19 data, helping users understand case distribution, daily trends, and country-level performance through dynamic visualizations.

## Poject Objective
The objective of this project is to analyze global COVID-19 data and transform raw datasets into an interactive dashboard for better understanding of pandemic trends and data-driven decision making.
## Tools Used
- Power BI
- Excel / CSV
- DAX
- Power Query

## Dashboard Pages
1. Home overwiew of Covid 19 report
- Overall summary of global COVID-19 statistics
- Total Active cases, confirmed cases, recovered cases, and deaths
- Interactive overview dashboard
2. Country Level data Trend Analysis
- Country-wise case comparison
- Recovery and death rate analysis
- Geographic distribution of cases using map visualization
3. Day Wise Analysis
- Daily confirmed case trends
- Daily death and recovery analysis
- Growth pattern visualization over time
